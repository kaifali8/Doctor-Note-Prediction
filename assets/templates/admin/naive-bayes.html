Naive Bayes Algorithm

Naive Bayes (NB) is a probabilistic classification algorithm based on Bayes' Theorem, assuming that the features are conditionally independent given the class label (the naive assumption). Despite this simplistic assumption, it works remarkably well in text classification and spam filtering.
